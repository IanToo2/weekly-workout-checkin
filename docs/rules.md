# Domain Rules

이 문서는 체크인 도메인의 규칙과 계산 기준을 정의한다.

## Core Rules

1. 1일 1회 체크인
- 동일 사용자(member)가 같은 날짜에 2회 이상 체크인할 수 없다.

2. 주간 목표 3회
- 주간 체크인 횟수가 3회 이상이면 달성으로 간주한다.

3. 주간 기준
- 한 주는 월요일 00:00:00부터 일요일 23:59:59까지로 본다.
- 주간 상태 조회 시 입력한 `date`가 속한 주를 기준으로 계산한다.

4. 미달 벌금
- 주간 체크인이 3회 미만이면 벌금 KRW 10,000
- 3회 이상이면 벌금 0

## Examples

예시 A)
- 주간 체크인 3회
- 결과: `passed=true`, `fineKrw=0`

예시 B)
- 주간 체크인 2회
- 결과: `passed=false`, `fineKrw=10000`

예시 C)
- 주간 체크인 0회
- 결과: `passed=false`, `fineKrw=10000`
