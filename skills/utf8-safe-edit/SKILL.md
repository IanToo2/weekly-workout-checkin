# UTF-8 안전 편집

한글이 있는 파일 수정 시 아래 규칙을 적용한다.

1. 기본 편집은 패치 기반으로 수행한다.
- 가능하면 `apply_patch`를 우선 사용한다.

2. PowerShell로 파일 저장 시 인코딩을 명시한다.
- `Set-Content -Encoding utf8`
- `Out-File -Encoding utf8`

3. 파일 저장 후 한글 존재 여부를 확인한다.
- `Get-Content -Raw -Encoding utf8 <path>`로 재조회
- 필요 시 `[가-힣]` 정규식으로 문구 보존 확인

4. 깨짐 발견 시 즉시 되돌리고 재저장한다.
- 깨진 상태로 추가 편집하지 않는다.

5. 다국어 UI 문자열 변경 시 의미 검증을 함께 수행한다.
- 버튼/에러/안내 문구가 의도대로 보이는지 확인한다.

## 출력 형식

```md
## 인코딩 점검
- 대상 파일:
- 저장 방식:
- UTF-8 확인:
- 한글 보존 확인:
- 조치 결과:
```
